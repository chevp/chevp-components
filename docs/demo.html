Themes & Palettes

2 base themes × 7 color palettes = 14 combinations. Click a swatch to preview.

Color Palettes

Usage

// Apply theme + palette import { darkTheme, paletteCobalt } from '@chevp/components'; document.adoptedStyleSheets = [darkTheme, paletteCobalt];

Variants

Panel

Container with header, icon, and action slot.

Basic

<chevp-panel heading="Tasks" icon="checklist"> Panel body content here. </chevp-panel>

Stat Card

Statistics display with label, value, and trend.

Variations

Profile Card

User profile display with avatar, name, and tags.

Basic

<chevp-profile-card name="Alice Chen" subtitle="Senior Engineer" color="var(--chevp-accent)" > <chevp-badge slot="tags" variant="blue">Frontend</chevp-badge> </chevp-profile-card>

Data Table

Configurable table with clickable rows.

Example

const table = document.querySelector('chevp-data-table'); table.setData( [{ key: 'name', label: 'Name' }, { key: 'role', label: 'Role' }], [{ name: 'Alice', role: 'Engineer' }] ); table.addEventListener('chevp:row-clicked', e => console.log(e.detail));

Task List

Interactive checklist with priority indicators.

Example

const list = document.querySelector('chevp-task-list'); list.setTasks([ { id: '1', text: 'Review PR #42', done: false, priority: 'high' }, { id: '2', text: 'Update docs', done: true, priority: 'low' }, ]); list.addEventListener('chevp:task-toggled', e => console.log(e.detail));

Timeline

Chronological activity feed.

Example

<chevp-timeline> <chevp-timeline-item icon="check_circle" icon-bg="var(--chevp-green-bg)" icon-color="var(--chevp-green)" time="2 hours ago" >Deployment successful</chevp-timeline-item> </chevp-timeline>

Schedule

Day-based schedule with time slots.

Example

const schedule = document.querySelector('chevp-schedule'); schedule.setDays([ { label: 'Monday', slots: [ { time: '09:00', text: 'Standup' }, { time: '14:00', text: 'Code Review', muted: true }, ]}, ]);

Bar Chart

Simple vertical bar chart visualization.

Example

const chart = document.querySelector('chevp-bar-chart'); chart.setData([ { label: 'Mon', value: 45 }, { label: 'Tue', value: 72, highlight: true }, { label: 'Wed', value: 58 }, ]);

Doc Editor

Rich text WYSIWYG editor with toolbar.

Example

<chevp-doc-editor placeholder="Start writing..."></chevp-doc-editor> // Get/set content const editor = document.querySelector('chevp-doc-editor'); editor.setContent('<p>Hello World</p>'); editor.addEventListener('chevp:doc-changed', e => console.log(e.detail.html));

Workspace

CSS Grid layout container for panels.

2-Column Layout

<chevp-workspace layout="2col"> <chevp-panel heading="Left" icon="dashboard">...</chevp-panel> <chevp-panel heading="Right" icon="analytics">...</chevp-panel> </chevp-workspace> // Preset layouts: "2col", "3col", "sidebar-main", "main-sidebar"
