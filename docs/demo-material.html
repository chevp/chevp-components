Installation

How to set up Material Design 3 components in your project.

1. Install
The MD variant includes @material/web as a dependency. No extra install needed.
View Code 
npm install @chevp/components
2. Apply Theme + MD3 Token Bridge
The mdTokens stylesheet maps your chevp design tokens to MD3 custom properties, so all @material/web elements inherit your active palette.
View Code 
import { lightTheme, darkTheme, paletteCobalt } from '@chevp/components';
import { mdTokens } from '@chevp/components/md';

// Light mode
document.adoptedStyleSheets = [lightTheme, paletteCobalt, mdTokens];

// Dark mode
document.adoptedStyleSheets = [darkTheme, paletteCobalt, mdTokens];
3. Use Components
Import the full bundle via @chevp/components/md or individual components. Same API as vanilla — just add -md to the tag name.
View Code 
// Full bundle
import { ChevpMdPanel, ChevpMdBadge } from '@chevp/components/md';

// Individual (tree-shakeable)
import '@chevp/components/md/panel';
import '@chevp/components/md/badge';
