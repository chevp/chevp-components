Data Table

Data table populated via setData(columns, rows). Emits chevp:row-clicked.

Basic Table
View Code 
<chevp-data-table id="table"></chevp-data-table>

<script>
  table.setData(
    [{ key: 'name', label: 'Name' }, { key: 'role', label: 'Role' }],
    [{ name: 'Alice', role: 'Engineer' }, { name: 'Bob', role: 'Designer' }]
  );
</script>

Task List

Interactive task list with checkboxes, priority indicators, and due dates.

Tasks with Priority
View Code 
<chevp-task-list id="tasks"></chevp-task-list>

<script>
  tasks.setTasks([
    { id: '1', text: 'Review PR', done: false, due: 'Today', priority: 'high' },
    { id: '2', text: 'Update docs', done: true, priority: 'medium' },
  ]);
</script>
